# Диагностика атмосферы молодежи — QR live

Это новая отдельная сборка. Она не использует и не изменяет старый код.

Файлы разделены специально:

- index.html — при открытии на ноутбуке автоматически ведёт в админку;
- host.html — только экран ведущего на ноутбуке;
- participant.html — только мобильный интерфейс участника, его открывает QR-код;
- host.js и participant.js — две независимые логики;
- app.css — стили;
- firebase-config.js и firebase-rules.json — подключение общей базы.

## Как это работает на встрече

1. На ноутбуке ведущий открывает адрес сайта. Откроется host.html.
2. На экране появляется отдельный QR-код конкретной сессии.
3. Участники сканируют его телефоном. QR открывает participant.html, а не админку.
4. Телефон просит никнейм и сразу запускает 16 вопросов.
5. На ноутбуке ведущий не отвечает на вопросы: он видит только подключения, кто отвечает и кто завершил.
6. Когда все завершили, ведущий нажимает «Получить личные результаты». У каждого на телефоне открывается личная карточка.
7. Ведущий нажимает «Показать общий экран». На ноутбуке появляется идеальная картина.
8. Через 30 секунд автоматически показываются реальные общие данные. Их можно показать раньше кнопкой.

## Один раз подключить Firebase

GitHub Pages размещает файлы, но без базы 20–30 телефонов не смогут работать как одна сессия. Для этого используется бесплатная Firebase Realtime Database.

1. Создайте проект в Firebase Console.
2. В Build → Authentication → Sign-in method включите Anonymous.
3. В Build → Realtime Database нажмите Create Database.
4. В Project settings добавьте Web app и скопируйте объект firebaseConfig.
5. Откройте firebase-config.js и замените объект на свой.
6. В Realtime Database → Rules вставьте содержимое firebase-rules.json и нажмите Publish.

## Загрузка в GitHub Pages

1. Создайте пустой репозиторий.
2. Загрузите в его корень все файлы этой папки.
3. В Settings → Pages выберите Deploy from a branch, ветку main, папку root.
4. После публикации открывайте обычную ссылку репозитория на ноутбуке. Участники входят только по QR с экрана ведущего.
